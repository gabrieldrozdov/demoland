Designing Interaction The humble button

The most commonly used animations on the web aren’t @keyframes animations, though. Instead, we encounter animation through the form of interaction.

We can make an element change its CSS properties by defining what’s called a “pseudo-class.” This is as simple as rewriting our CSS class and adding :hover at the end of the class name. Now, whenever we hover over our HTML element, its CSS will change!

One last important detail: to animation between states, we need to add the transition property to our main CSS class.

CSS Properties

Here are the CSS properties used in this demo:

  • border
  • border-radius
  • font-family
  • padding
  • font-size
  • margin
  • transition
  • cursor
  • background-color
  • color
