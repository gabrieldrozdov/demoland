Basics What is HTML?

HTML (Hypertext Markup Language) is the main programming language of the web. It organizes information using HTML elements, which are marked by opening and closing HTML tags. For example, a paragraph opens with <p> and closes with </p>.

The code in the Editor panel is all HTML. Notice how each HTML element gets interpreted by the browser in the Preview panel. HTML tags (colored red in the Editor panel) are not visible on a live website. The Editor panel (and any text editor) will automatically color your code to let you know if it contains any typos.

By itself, HTML can’t do much other than organize information. To actually control the appearance of HTML elements, we’ll need to use CSS.
