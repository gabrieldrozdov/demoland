Basics What is JavaScript?

While HTML and CSS let us lay out and style information, JavaScript (or JS) gives our site deeper functionality. This functionality has a wide range of possibilities, from creating basic text pop-ups (like the one featured in this demo) to much more complicated interactions (like this entire site’s interface and code editor).

JavaScript is very different from HTML/CSS and works more like a traditional programming language. That means that what you code in JavaScript is also codable in virtually any other programming language. Meanwhile, HTML and CSS are unique to the medium of websites. JavaScript’s syntax is also substantially more complicated than HTML and CSS’s.

We can write JavaScript inside of the <script> element, which we typically place at the bottom of our document. We can also use inline event handlers inside of HTML elements to call JavaScript functions.

Be sure not to confuse JavaScript with Java, which is an entirely unrelated programming language! Java was very popular, so the creator of JavaScript named it after Java to try to bank on that popularity.
