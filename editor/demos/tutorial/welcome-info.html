Welcome to DEMOLAND

DEMOLAND is a collection of demos presented right inside your web browser. Learn about, edit, and write your own HTML, CSS, and JavaScript without the need to download any additional software!

Hover over any button to see what it does, and read below to learn about everything DEMOLAND can do.

Overview

DEMOLAND’s interface features three panels:

  1. The Info panel features information about the demo you’re viewing.
  2. The Editor panel lets you see and edit the demo’s code.
  3. The Preview panel shows you a live preview of the code.

Selecting a Demo

Navigate between demos using the menu at the top of the Info panel.

Access the catalog or this tutorial via the buttons at the top-left.

Reset the current demo, download its code, or open the demo in a new tab using the buttons at the bottom-left.

Customizing the Interface

Hide/show panels by using the buttons to the center-left.

Resize panels by dragging between them. The bottom of the Preview panel displays its width and height.

The bottom of the Editor panel features controls for wrapping lines of code and decreasing or increasing the font size.

Writing Code

Write your code directly in the Editor panel. By default, code is interpreted as HTML.

To write CSS, use the <style> element.

To write JavaScript, use the <script> element.

Your code will generate a live preview in the Preview panel. On the bottom of the Editor panel, you have controls to delay, pause, or refresh this preview.

The Editor panel also features a console for testing your JavaScript code. This console is not a full substitute for your browser’s console, but still works in most cases.
