Welcome to DEMOLAND

DEMOLAND is a collection of coding demos that are editable right inside your web browser. Learn how to code without the need to download any additional software!

Interface

Info panel

The first panel, which you’re currently reading. This panel provides information about your current demo.

Editor panel

The second panel, which is where you can view and edit code. The bottom of this panel lets you customize the Editor’s interface, as well as control how often your code gets previewed.

Preview panel

The third panel, which displays a live preview of your code. The bottom of the Preview panel displays the current dimensions of the Preview panel.

Customizing the interface

Hide/show panels using the three buttons to the center-left. Resize panels by dragging between them.

Navigation

Selecting demos

Select demos in the current collection by clicking the menu bar at the top of the Info panel.

Opening a different collection

Return to the homepage by clicking the “Catalog” button in the top-left corner.

Controlling the current demo

Reset the current demo, download its code, or open the demo in a new tab using the buttons at the bottom-left.

Need help?

You can reopen this tutorial collection at any time with the button underneath the “Catalog” button in the top-left corner.

Writing Code

Write your code directly in the Editor panel.

HTML

By default, all code is interpreted as HTML.

CSS

Write CSS using the <style> element.

JavaScript

Write JavaScript using the <script> element.

Open the next demo in this collection to learn more about how to write HTML, CSS, and JavaScript!
