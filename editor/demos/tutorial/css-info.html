Basics What is CSS?

CSS (Cascading Style Sheets) is a programming language that lets you control the appearance of HTML elements. Its code looks a little different than HTML (this is called its syntax), but youâ€™ll often find it right next to HTML.

CSS consists of properties organized into classes.

Inline CSS lives directly inside of HTML tags via the style attribute. Alternatively, we can reuse CSS by writing classes inside the <style> element, which we typically place at the top of our document. In either case, the Editor panel will interpret your code as CSS, not HTML.
