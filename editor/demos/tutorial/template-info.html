Getting Started Starter Template

This template features all the basic things you need to get started making a website!

Write your HTML inside the <body> element, stopping before the <script> tag.

Write your CSS inside the <style> element, which lives inside the <head> element. The <head> contains invisible metadata about your website and always comes first.

Write your JavaScript inside the <script> element, which lives right at the end of the <body> element. JavaScript usually needs to reference the HTML elements in your code, which means it has to come after you’ve actually written those HTML elements.
