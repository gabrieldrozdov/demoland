Paragraphs support some inline (within the text itself) elements. We can use the “em” (or emphasis) tags to create italicized type. Then, there’s the “strong” tag for bolded type. Also, superscript and subscript type.

There’s also the span tag, which does nothing on its own but will be helpful when we start working with CSS.

We’re not allowed to type some characters because our code requires them, like greater than or less than symbols. We can use HTML entities to get around this so that we can type things like <p> without creating a new paragraph element!
