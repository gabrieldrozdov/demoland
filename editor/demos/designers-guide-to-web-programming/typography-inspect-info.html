Typography Using the Inspect Tool

Web browsers have helpful tools built in to analyze and debug websites. One of those tools is the inspect tool.

Please use Google Chrome for this demo!

Shortcuts: MacOS: COMMAND+OPTION+C
Windows: CONTROL+SHIFT+C

You can also right-click anywhere on a page and then click “Inspect”.

Once in the inspect tool, click the button in the top-left (it looks like a square with an arrow pointing at the center). This tool lets you highlight elements on a page to see their properties.

With an element highlighted, view the “Styles” and “Computed” tabs to see what CSS is applying to that element!
