By default, images are NOT responsive. That means they won’t work well on different screen sizes (especially on mobile devices). Why? Here’s a few reasons:

  • Images by default are inline, not block, elements
  • Images by default don’t have maximum widths (or set widths at all)

To fix this, we can set images to have a width of 100% and a display value of block. The image’s height will figure itself out.
