Type as Visualization Setting Values Using Sliders

Sliders let us predefine a range of values that we can then use to set variable axis values, as well as other CSS properties. We can create a slider using <input type="range" min="_" max="_" value="_">, where “min” is the minimum value, “max” is the maximum value, and “value” is the default value.

With the oninput="_" event, we can run a function every time the slider gets changed. In this case, we have a function for each slider. These functions update the corresponding CSS variable, which updates our live text.
