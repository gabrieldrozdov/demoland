The Box Model CSS Overrides

Some HTML elements have default CSS properties that can be annoying to work with. For example, paragraphs have a default margin, and all elements have box-sizing: content-box when we’d rather have box-sizing: border-box.

We can easily override the defaults using the special asterisk * CSS selector. This applies CSS to every HTML element on the page.

You can change whatever defaults you want, but I like to set every element’s margin and padding to 0, and then fix the box-sizing property.

Fun fact: the <body> element has a default margin! It’s annoying!
