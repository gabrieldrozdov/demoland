CSS CSS In-use

There’s a wide range of ways to use CSS depending on the context. Here’s a few more things you should know:

Multiple CSS classes

You can apply multiple CSS classes to an element by adding a space between class names inside of the class attribute.

When to use inline CSS

Inline CSS is useful for one-off styling. You can apply it inside of a paragraph or other text element using the <span> tag.

Also, inline CSS will override CSS from classes.

Global CSS

If you write a CSS class in the style element but don’t start with a period, then that CSS will actually apply to all elements with that name. So in this demo, we’re styling all li (list item) elements for example.
