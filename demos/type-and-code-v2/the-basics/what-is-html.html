This is a basic demo of HTML!

HTML is nifty! Here’s what it can do:

  • Organize information, like text or images
  • Tell the web browser what kind of information something is, like a heading, paragraph, or list
  • Provide metadata to a web browser for sharing your site (like the thumbnail you see when texting someone a URL)

Lines of HTML code can get pretty long. To make all of your code fit on the screen, click on the line wrap toggle at the bottom of the Editor panel.

HTML also hijacks some characters because it needs them to write code. For example, I can’t write a “less than” sign without creating a new element. So, I need to use what’s called an HTML entity. We can search for these online, but here’s the one for the less than sign <

We can put some information inside of HTML elements to give them more functionality. These are called HTML attributes and live inside the opening tag of HTML elements. For example, links use the <a> element and take an “href” attribute to decide what link to navigate to. You can also use the “target” attribute to control if the link opens in a new tab or not!
