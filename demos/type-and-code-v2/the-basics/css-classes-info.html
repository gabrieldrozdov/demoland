CSS Classes

In many cases, we want to apply the same CSS to mulitple HTML elements. We can do that using CSS classes.

We write CSS classes inside the <style> element, which we typically place at the top of our document. If our CSS is not inside of the <style> element, then the browser will interpret it as HTML and the website won’t render properly.

To indicate a class in the <style> element, we write a period (.) followed by the class name (all lowercase, no spaces!) and then curly brackets {}. Inside the curly brackets, we can write our CSS properties and values, making sure to add a semi-colon (;) after each value.

Classes also make it easier to separate our CSS from our HTML, so that our code is easier to read!
