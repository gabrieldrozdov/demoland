Here’s a demo to help you practice all of the concepts we’ve talked about so far. In the demo to the right, you’ll find some text from an article (Dogs from Wikipedia). Try to complete the following tasks:

  1. Organize the article contents using HTML elements.
  2. Add CSS classes to those elements to style them, using CSS properties for typography and the box model.
  3. Style specific sentences, words, or letters by adding span elements and using inline CSS.
