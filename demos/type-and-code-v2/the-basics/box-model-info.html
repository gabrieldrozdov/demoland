Box Model

The last thing worth knowing for now is called the CSS Box Model. This is the structure that determines how CSS styles HTML elements.

Basically, every HTML element is a box, with several CSS properties controlling how that box looks and gets positionined on the page.

Click here to see all the CSS properties used to style the box model!

The * selector and the box-sizing property

The * selector applies CSS to every single HTML element. There is a weird property called box-model whose default value messes up the width property. For literally every website I make, I set box-sizing for all elements to the value border-box. Click here to learn about the box-sizing property!

A note on margin

Margin is weird, in that margins between elements will overlap on each other instead of adding up. So if two elements have a margin of 40px, then there will only be 40px of space between those elements (as opposed to 80px).
