Putting It All Together

In class, we saw that we could type ! and then press enter in Visual Studio Code to get an HTML template. That template helps separate out the HTML and CSS portions of our document.

The style element (where we write CSS classes) lives inside the head element. This element contains all the invisible metadata for our document.

Meanwhile, all our visible HTML elements (like typography) live inside the body element.

In class, we’ll also look at how we can put our CSS in a separate document entirely to keep ourselves even more organized.

To learn about all the elements used in the standard HTML template, click here! You don’t actually need to know what all of them do, but you need to include them in all of your projects.
