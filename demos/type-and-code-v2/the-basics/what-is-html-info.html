What is HTML?

HTML (Hypertext Markup Language) is the main programming language of the web. It organizes information using HTML elements, which are marked by opening and closing HTML tags. For example, a paragraph opens with <p> and closes with </p>.

The code in the Editor panel to the right is all HTML. Web browsers interpret HTML code and turns it into real websites. For example, HTML tags (colored red in the Editor panel) are not visible on a live website. The Editor panel (and any modern text editor) will automatically color your code to let you know if it contains any typos.

HTML only handles content, so to design (or “style”) that content we’ll need to use another programming language called CSS. Open the next demo to learn about CSS!
