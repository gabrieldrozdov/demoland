Info How to generate a random number

(This isn’t a challenge, but you’ll need to know how to do this later on!) Here’s how you generate a random number in a given range. Let’s use the range of -43 to 229 as an example.

  1. Create a random number using Math.random(). This number will be a decimal (or to use a technical term, a float) number between 0–1.
  2. Multiply this number by full range of numbers you want. To get the range, we subtract the minimum by the maximum, so 229-(-43)=272. This will give us a number between 0 and 272.
  3. Then, add the minimum value to this result. So, if we got 0 in Step 2, our number will now be 0+(-43)=-43. If we got 272 in Step number, our number will be 272+(-43)=229. That means we can only get numbers within the correct range!
  4. If you want to, round your number.
    You can round using Math.round().
    If you want to always round down, use Math.floor().
    If you want to always round up, use Math.ceil().
