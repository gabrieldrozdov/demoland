Challenge Variable Axis Sliders

This demo uses CSS variables to make a functioning variable axis slider. Hereâ€™s how it works:

  1. The sliders are preset with the variable axis value ranges. So, the weight axis goes from 100 to 900, and the serif axis goes from 0 to 100.
  2. When the sliders are changed, they run a function that takes three inputs. The first is the selector (like ".class" or "#id") for what element to target. The second is the variable name (like "--some-name") for the CSS variable to change. The third is the slider value itself to set that variable to.
  3. Then, it targets the element for the given selector and updates its CSS variable.
