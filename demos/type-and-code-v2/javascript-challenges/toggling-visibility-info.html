Challenge Toggling Visibility

In this challenge, I’ve coded a menu that defaults to being hidden. There’s a openMenu() function that shows the menu.

Add a closeMenu() function that hides the menu when the menu-close element is clicked. Do the following:

  1. Add the onclick attribute to the button to call the function.
  2. Create the function in the JavaScript portion of the code.
  3. Inside the function, target the menu and add the hidden class.
