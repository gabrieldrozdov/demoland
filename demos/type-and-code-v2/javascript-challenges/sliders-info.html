Challenge Sliders

Sliders are useful for dynamically updating CSS properties. In this example, the slider has a few built-in settings in the HTML element:

  • Minimum value (min="0")
  • Maximum value (max="100")
  • Default value (val="0")
  • A function that runs when the slider updates (oninput="updateSlider(this.value)")

This this.value text means to run the function using the slider’s current value.

The function then takes that value and outputs it to a paragraph.

Your Task

Update the paragraph’s font size to match the slider’s value.
