The Box Model Box Sizing

The box-sizing property affects how width is calculated. By default, an element’s padding and border are not included in its overall width. That means that setting an element to a width of 100% causes it to overflow its container.

To fix this, we can use box-sizing: border-box. Easy!

Why is this not the default setting? Long story short, the Internet is old. This is how it was originally built, and nowadays we don’t want to break how old websites work, so we have to override their default settings.
