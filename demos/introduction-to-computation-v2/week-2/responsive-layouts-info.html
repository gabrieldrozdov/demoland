Responsive layouts

The size of a website depends on the browser window and the user’s device. This means that our websites need to be responsive and change according to screen size.

With the box model, we can specify an element’s width using the width property. But, we want that width to change depending on the screen size. Here’s how we can do that:

  1. Set the element’s width value to 100%. This means it will take up the full width of its parent element.
  2. Set the element’s max-width to a pixel value like 600px. This means that the element will be no wider than 600px across.
  3. Set the element’s left and right margins to auto. This pushes the element to the middle of the screen.

The box-sizing property

By default, the width property doesn’t include padding in its calculation. That means that an element may be too big for its parent. We can fix this by writing box-sizing: border-box;
