CSS inheritence

We can nest HTML tags inside of other HTML tags.

  • Parents are the outer elements.
  • Children are the nested elements.
  • Siblings are elements at the same nesting level. In this demo, the paragraphs are all siblings of each other.

Some CSS properties pass down from parents to children. This is called CSS inheritence, where the children inherit CSS properties from their parents.

Not all properties are automatically inherited, but most properties relating to typography are inherited.

The div element

Sometimes, we need a generic HTML element. Here’s why this might be the case:

  • To group other elements together.
  • To create an element that doesn’t have a prescribed type of content.

In this demo, we use the generic <div> element to group our stanzas together.

CSS comments

We can add notes inside of our CSS code using CSS comments. Comments start with /* and end with */
