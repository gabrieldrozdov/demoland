HTML elements for typography

When typesetting on the web, we use a few common elements that differentiate different kinds of text. Learn more about these elements on Cheatsheet.

HTML comments

We can add invisible annotations to our HTML code using HTML comments. We start a comment by writing <!-- and end a comment by writing -->.

You can quickly comment or uncomment highlighted text by pressing COMMAND+/ on MacOS or CONTROL+/ on Windows.

Block and inline elements

Most elements take up the full width of the page and result in a line break, like paragraphs. These are called block-level elements.

Some elements live inside of other elements and just take up the space their content takes up, like emphasized (italicized) text. These are called inline elements.

Nesting

We can put HTML elements inside of other HTML elements. This is called nesting. Nesting elements results in the following relationships:

  • Parents are the outer elements.
  • Children are the nested elements.
  • Siblings are elements at the same nesting level. In this demo, the em and other inline elements are siblings of each other.

HTML attributes

HTML attributes provide extra information to HTML elements.

Where to add attributes

Attributes live inside the opening tag of an element. For example, a paragraph using the opening tag <a>. We would add any attributes after the tag name a and before the > symbol.

Formatting attributes

Attributes use the format name="_", where “name” is the name of the attribute. The formatting of text inside the quotes depends on what attribute we’re using.

The “href” and “target” attributes

For link elements, the href attribute indicates the URL to navigate to.

Adding target="_blank" makes that link open in a new tab when clicked.
