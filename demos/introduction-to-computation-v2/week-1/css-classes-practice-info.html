The body element

The body element represents all the visible content on a website. Every web page should have exactly one body element.

We can add CSS to the body to style the full page.

CSS selectors

In this demo’s style element, notice how we wrote body with no leading period, and .stanza1 with a leading period. These are different kinds of CSS selectors.

  • Class selectors start with periods. Classes only affect elements that have the class assigned inside the element’s opening tag.
  • Element selectors use the element’s name without a leading period. These styles will affect all elements of the same name.

span elements

span elements are generic inline elements that we can use to style individual words or lines of text.
