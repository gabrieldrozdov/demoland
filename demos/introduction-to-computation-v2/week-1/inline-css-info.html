Styling content using CSS properties

CSS (Cascading Style Sheets) is a second programming language used alongside HTML. While HTML handles content, CSS handles design (a.k.a styling).

CSS properties and values

CSS features many distinct properties that affect different aspects of an HTML element’s design. For example, font-family determines the element’s font.

We assign values to each property. We do this by following the property name with a colon :. For example, we can write font-family: monospace to use the default monospace font.

Using multiple properties

We separate CSS properties with a semicolon ;. For example, font-family: monospace; color: blue;

The “style” attribute

We can apply CSS to an HTML element by using the style="_" attribute. Everything inside of the quotes will be interpreted as CSS.

This is called inline CSS.
