Positioning

By default, every new element will appear below the previous element. The only exception are inline elements (like italicized or bolded text) that appears inside other elements.

To break out of this flow, we can use the position property. This property has five possible values:

  • static
  • relative
  • sticky
  • absolute
  • fixed

We typically use position property in conjunction with the following properties:

  • top
  • right
  • bottom
  • left

position: static;

This is the default position value. Elements will flow as per usual.

position: relative;

The element will occupy its usual position, but you can visually move it using the top and left properties.

position: sticky;

The element will flow just like a static element, but once it reached a certain position will stick to its location on the screen. That position is usually defined by the top property.

position: absolute;

The element breaks the flow of the page entirely and can be positioned manually using the top, left, right, and bottom properties.

For example, the top property indicates the position relative to the top of the page.

You can also position an element inside of another one using absolute positioning. This works if the parent has its position set to either relative or absolute.

position: fixed;

The element breaks the flow of the page entirely and can be positioned manually just like with the absolute value. Except, this element will stay in the exact same location no matter how the page scrolls.
