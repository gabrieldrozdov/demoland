HTML entities

If we want to write certain characters like the < symbol, we have to use special codes called HTML entities. Otherwise, the web browser will interpret these symbols as the start of HTML elements. For example, we can write < by saying &lt; (which stands for “less than”).

In this poem, we’re using the non-breaking space entity, which lets us write multiple spaces in a row.
