Advanced CSS selectors

So far, weâ€™ve seen class, element (.), and id (#) selectors. There are dozens of selectors we can use, which you can find a full list of on MDN. Here are a couple of useful ones:

  • The * selector applies CSS to every element.
  • By writing two selectors, like div > div, we can apply CSS to children of specific elements.
  • We can target every other child using the :nth-child(_) selector. This selector supports a variety of equations beyond just even or odd.
  • And, we can combine any of the above selectors together to make even more complex selectors.
