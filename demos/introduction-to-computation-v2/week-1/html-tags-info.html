Organizing content using HTML tags

HTML (Hypertext Markup Language) is the primary programming language of the web. We use HTML to organize content in our plaintext files so that it displays correctly on websites.

Code linting

To make your life easier, text editors (applications used to edit plaintext files) will dynamically color your code to indicate if you’re following the correct syntax. This is called linting.

Try deleting the first > symbol in the text to see how the linting indicates there’s an error.

HTML elements

HTML elements are blocks of content of a specific kind. We can create HTML elements by marking their start and end with HTML tags.

Notice that some of the text in the “Editor” panel is not visible in the “Preview” panel. This is invisible code that helps web browsers understand what content is in an HTML file, and how to display that content.

HTML tags

Opening tags use the format <name>, where “name” refers to the element we’re creating.

Closing tags use the same format with a forward slash, i.e. </name>.

For example, paragraphs start with <p> and end with </p>.

Most elements require both an opening and closing tag, but a few elements only require the opening tag. One example is the line break element <br>.
