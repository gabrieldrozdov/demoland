Organizing styles using CSS classes

In the previous demo, we had to use inline CSS on every element to change the website’s design. If we wanted to change the font or color across all elements, we would need to update every element individually.

CSS classes let us organize our CSS styles separately from our HTML.

Defining classes

We write our CSS at the top of our HTML file in the <style> element. Everything inside of this element will be interpretted as CSS, not HTML. This is called internal CSS.

Class names

To indicate a class, we write a period . following by a class name. This name should be in lowercase and cannot use spaces or special symbols. If we want a class name with multiple words, we’d separate the words with hyphens -, for example .highlighted-stanza.

Adding properties

After our class name, we write our CSS property-value pairs inside of curly brackets {_blank}.

Reminder: after every property-value pair, make sure you add a semicolon ;

Using classes

To assign a class to an HTML element, we use the class="_" attribute. You can assign multiple classes to an element by adding a space between the class names.

Note: there must only be one class="_" attribute per element.
