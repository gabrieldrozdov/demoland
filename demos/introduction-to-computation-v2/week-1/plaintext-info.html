Plaintext files

Code is just text. We store that text in plaintext files. These are files that contain text without formatting. That means this text has no prescribed fonts, colors, etc.

Programming languages

Depending on what we’re coding, we have to select a programming language. When we’re making websites, we use three programming languages: HTML, CSS, and JavaScript.

When we save a plaintext file, we use the file extension to indicate what language we’re writing in. HTML uses the file extension .html.

Try downloading this demo (the code in the “Editor” panel) by pressing the “Download Code” button in the bottom-right corner to see a .html file!

Syntax

Each programming language has a set of rules we need to follow so that a computer can understand what we’re writing. This is called the language’s syntax.

The “Editor” panel to the right shows HTML code representing William Carlos Williams’ poem “The Red Wheelbarrow.” The “Preview” panel shows how this code will appear on a live website. Notice how HTML’s syntax ignores line breaks when interpreting code.
