CSS box model

On the web, every element is a box. The styling of these boxes uses a structure called the CSS box model.

The box model mainly consists of the following CSS properties:

  • margin, the space surround an element
  • padding, the space inside an element
  • border, a line between the margin and padding
  • background-color, self-explanatory
