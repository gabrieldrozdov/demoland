Geometry + Material = Mesh

To put a visible object in the scene, you need three things combined:

Geometry

The shape — a mathematical description of vertices and faces.

const geo = new THREE.BoxGeometry(
  1, // width
  1, // height
  1  // depth
);

Material

The surface — how the shape looks (color, shininess, etc.).
MeshBasicMaterial ignores lights. MeshStandardMaterial reacts to them. 

const mat = new THREE.MeshStandardMaterial({
  color: 0xff6b35 // hex color
});

Mesh

Combining geometry + material into an object you can place in the scene.

const cube = new THREE.Mesh(geo, mat);
scene.add(cube); // ← don't forget this!

Lights

MeshStandardMaterial needs at least one light or the object will appear black.

const light = new THREE.DirectionalLight(0xffffff, 1);
light.position.set(2, 4, 3);
scene.add(light);

// AmbientLight fills in shadows so nothing goes pitch black:
scene.add(new THREE.AmbientLight(0xffffff, 0.4));
