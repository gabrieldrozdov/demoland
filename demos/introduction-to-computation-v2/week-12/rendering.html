The 3 things every Three.js page needs

1. Scene

The scene is the "world" — an empty container you add objects to.

const scene = new THREE.Scene();

2. Camera

The camera defines your viewpoint. PerspectiveCamera is the most common — it mimics a real camera with a field of view and perspective.

const camera = new THREE.PerspectiveCamera(
  75,           // field of view (degrees)
  width/height, // aspect ratio
  0.1,          // near clipping plane
  1000          // far clipping plane
);
camera.position.z = 5; // move back so we can see

3. Renderer

The renderer draws the scene using WebGL. You give it a canvas element (or let it create one) and call render().

const renderer = new THREE.WebGLRenderer();
renderer.setSize(width, height);
document.body.appendChild(renderer.domElement);

// Draw one frame:
renderer.render(scene, camera);

Right now, the canvas shows…

Nothing! The scene is empty. That's expected. We have the three pieces working — next step is adding an object.

  innerWidth/innerHeight, 0.1, 1000);
