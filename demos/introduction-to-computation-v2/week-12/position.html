Position, Rotation & Scale

The coordinate system

X →
Y ↑
Z ● toward you

Three.js uses a right-handed coordinate system. The AxesHelper in the scene shows this — red=X, green=Y, blue=Z.

position

cube.position.x = 2;   // move right
cube.position.y = 1;   // move up
cube.position.z = -3;  // move away
// or all at once:
cube.position.set(2, 1, -3);

rotation

cube.rotation.x = Math.PI / 4; // 45°
cube.rotation.y = Math.PI;     // 180°
// rotation is in RADIANS, not degrees
// Math.PI = 180°  |  Math.PI/2 = 90°

scale

cube.scale.x = 2;    // twice as wide
cube.scale.y = 0.5;  // half as tall
cube.scale.set(1, 1, 1); // back to normal
