Week 1 Making it pretty

CSS, which stands for Cascading Style Sheets, is the programming language that styles HTML elements. You can think of HTML as content, and CSS as style.

What makes two programming languages different? They use different syntax, which are the rules we have to follow when writing code in a particular language. HTML uses elements indicated by opening and closing tags. CSS using properties with values, organized into classes.

Since HTML and CSS work so closely together, we can actually embed CSS into an HTML document. We do that using the <style> element. Everything within this element must follow CSS syntax.

CSS syntax follows this format:

  • A class name that starts with a period. Class names cannot contain spaces and should only contain hyphens or underscores (- or _) for punctuation.
  • Curly brackets indicating the contents of that class, i.e. {}
  • Property names followed by colons :
  • Values followed by semicolons ;

We can then apply a CSS class to an HTML element via the HTML “class” attribute. Attributes live inside the opening tag for an HTML element and are followed by an equals (=) sign and quotes (" ") with the class name.
