Week 1 Programmatic poetry

It’s time for your first pair programming challenge!

In this demo, you’ll find the William Carlos Williams poem “The Red Wheelbarrow” written as HTML elements. In groups of two (or three), your task is to style the poem using the CSS class stubs (placeholders) provided in the document. One person is going to be the main coder, while the other person (or people) will offer over-the-shoulder guidance. Try to make the poem feel as “poetic” as possible.

Below, you’ll find a list of useful CSS properties for styling HTML elements in general and also for specifically styling typography. Click on each property to see its relevant MDN page, which documents how to use that property with examples. This is the official resource that all web developers use to learn how different properties work.

Notice that this document has a new HTML element — the <body> element. All visible content on a website lives inside of this element. For now, we can use it to give the page a color or a margin. We’re also using <br> tags to add line breaks — notice that these elements don’t feature closing tags!
