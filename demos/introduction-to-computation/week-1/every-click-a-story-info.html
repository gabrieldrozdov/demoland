Week 1 Every click, a story

OK, so we used HTML and CSS to make a poem look pretty and to make a square dance. We haven’t talked about the most important thing that code is capable of — making things interactive.

In this demo, you’ll find a new HTML element — the <button> element. Buttons are just one of dozens of HTML elements that exist. We’re learn about all of them in due time.

CSS makes it really easy to add basic interactivity to a website. By duplicating a CSS class and writing :hover after the class name, we can create a hover state. This hover state changes the element’s CSS when you hover over it. Similarly, we can repeat this process with :active to create an active state (when you’re pressing down on the element). Lastly, if we give the main class the transition property, we can even have these states animate between each other!

For this pair programming challenge, make one button extremely satisfying to press, and the other button extremely unsatisfying to press. You can pick which one is which, but don’t tell us — we’re going to guess!
