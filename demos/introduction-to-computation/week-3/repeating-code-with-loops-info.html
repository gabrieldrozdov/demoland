While Loops Repeating code with loops

JavaScript is capable of running a large amount of code instantenously. If we want to automatically run some code multiple times, we can use loops.

In this demo, we have a while loop. while loops look like if statements and similarly check if a condition is true. The loop keeps running the code inside of its curly brackets until the condition becomes false.

We’ll frequently want to run a loop a certain number of times. To do that, we’ll want to create a variable we can use as a loop counter. In this demo, we have the variable flips which we initialize with the value 0. Every time the loop runs, we increase flips by 1. The loop stops running once flips is greater than or equal to 10.

Loops can be dangerous because it’s easy to end up in a state where the loop condition never becomes false. In this case, we end up with an infinite loop. Infinite loops will cause your code to break.
