innerHTML Visualizing results

Functions become much more useful once we start tracking and displaying their results (i.e. whatever they return, or any of their side effects). We can easily do this with a little HTML.

This is the same demo as before, but we have a new <div> element. This element has an id attribute, which we can use as a unique identifier.

We can then target our element in our JavaScript. This lets us change its properties, such as its content and styling. We target elements by using document.querySelector('#[element-id]'). We can then assign the targeted element to a variable to easily access it in our JavaScript.

In this demo, we target the element with the ID of “results” and then change its content using .innerHTML. We can set this property to any string, which gets interpreted as actual HTML.
