Week 2 Scope

The scope of a variable refers to the part of the program where the variable is accessible and can be used. JavaScript has a few different scopes you should know:

  • Global scope, declared outside of any function or block
  • Function scope, declared inside a function
  • Block scope, declared inside a block (i.e. anything featuring curly brackets { } )

Any variable defined within a function or block is called a local variable.

Variables defined using var ignore block scoping. That means we can define global variables even in a function as long as we use var.
