Week 2 Data types

There are five essential data types worth knowing:

  • Numbers (either integers or floats)
  • Strings (text)
  • Booleans (true or false)
  • Arrays (a.k.a. lists)
  • Objects (a.k.a. dictionaries)

Micro challenge

Create an object with the following keys:

  • "name" (a string)
  • "age" (a number)
  • "courses" (an array of strings representing your course names)
