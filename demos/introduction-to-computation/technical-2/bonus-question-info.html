Technical 2 Bonus Question

About this demo

This demo features a button. When the button is clicked, a function called readAllData runs.

The JavaScript portion of this demo features an array called myData. This array contains 4 other arrays, each of which contains several strings. When the readAllData function runs, it triggers 4 alert messages. Each alert contains the data from one of the sub-arrays of myData.

If that’s confusing, just click the button to see what happens!

Your task

Edit the function so that each string gets its own alert message instead of being grouped into an array. For example, right now the alert for the first sub-array looks like the following:

  • red,green,blue

We want it to look like this instead (3 separate alert messages):

  • red
  • green
  • blue

To do this, you’ll want to make a for loop inside of the pre-existing for loop.
