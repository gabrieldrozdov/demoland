Input Elements Building an interface

Now that we have the ability to run our function multiple times, we can let the user have more control over the number of loops. We can add this functionality using <input> elements. (Note that these elements don’t have closing tags.)

There are a variety of <input> element types we can use, which we specify using the type attribute. In this demo, we’re using an input that only allows for the user to input numbers. We set the element’s default value using the value attribute.

We can get then get the current value of our <input> element in our JavaScript. To do so, we target our element, and then grab that element’s value property. This returns the element’s number.
