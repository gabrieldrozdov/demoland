For Loops Different kinds of loops

The previous demos used while loops. for loops are an alternative kind of loop we can also use.

for loops use a similar construction as while loops. Except, the condition (i.e. the text within the parentheses) has three distinct statements separated by a semicolon:

  1. Initialization for a loop counter variable
  2. Condition that gets evaluated each time the loop runs
  3. How the counter changes each loop

for loops can be easier to keep track of than while loops because they contain all the information they need in a single line.
