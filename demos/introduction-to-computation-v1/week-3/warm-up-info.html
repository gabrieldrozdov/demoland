Warm Up Poem generator

Today’s lesson is about using loops to run code repeatedly. Here‘s a warm up to introduce some of the concept’s we’ll learn in today’s class.

The JavaScript for this demo features several arrays containing words. These arrays are used to generate a random poem.

Your task

Customize the words in the following arrays to generate a poem on your choice of theme or topic:

  • articles
  • adjectives
  • nouns
  • adverbs
  • verbs
  • conjunctions
