Solving Problems Helper functions

It’s often useful to have secondary functions that help our main functions run. These are functions that complete intermediary steps before we return or output our final result. We call these secondary functions helper functions.

In this demo, we’re using a helper function to randomize (or shuffle) an array. I didn’t write this function myself and instead found it here.

Some helper functions are built-in to JavaScript. For example, if you have an array of numbers myNumbers, you can sort that array by saying myNumbers.sort(). Another example is the trim function, which removes spaces at the start and end of a string.
