Math.random() Getting random values

We’ve seen how Math.random() gives us a random float between 0 and 1 (not including 1, so really up until 0.99999...). We can use this random number generator to do two more things:

  • Generate a random integer in any specified range
  • Use that integer to grab a random value from an array

In this demo, we have two helper functions. The first helper function is called getRandomInteger and generates a random integer (rounded number) between an inputted minimum and maximum value. It then returns that integer. Here’s how it works:

  1. Calculate the total range of possible numbers by subtracting the minimum from the maximum (max-min).
  2. Generate a random float between 0–1 using Math.random().
  3. Multiply the random float by our full range from step 1.
  4. Add the minimum possible value to that number.
  5. Round that number down using Math.floor().

Example: Generate a random number between -100 and 50

  1. First, we subtract the max (50) by the min (-100). This gives us 50 - -100 = 150.
  2. Math.random() gives us a random number between 0 and 1.
  3. 0 * 150 = 0 and 1 * 150 = 150. That means that multiplying our random float by our range gives us a random number with a minimum of 0 and a maximum of 150.
  4. 0 + -100 = -100 and 150 + -100 = 50. That means we’ve officially ended up with a number in the range of -100 and 50! But this number is still a float
  5. We round that number down because if we’re accessing a value from an array, we don’t accidentally want to get a number too high for the array’s range. We could also round that number up using Math.ceil() or to the nearest integer using Math.round()

The second helper function is called getRandomValue. This function takes an array as its input and then uses getRandomInteger to generate a random index in the array’s range. It then returns that value.
