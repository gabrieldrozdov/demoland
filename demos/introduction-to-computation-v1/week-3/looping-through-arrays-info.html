for...of Looping through arrays

We can use our knowledge of array indexing to loop through arrays automatically. Here are three common methods for doing so:

  • Using a while loop and a variable that keeps track of the current index
  • Using a for loop that does the same thing
  • Or, using a for...of loop, that lets us reference the contents of our array directly
