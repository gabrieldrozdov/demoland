JavaScript Events Running functions

So far, we’ve run functions within the script element. For example, if we had a function called isEven that took an argument, we could run it with isEven(2), which might return true.

We can also run functions based on user input, like when a button gets pressed. This is accomplished using event handlers. Event handlers can detect different user inputs, like clicks and scrolls, and then run JavaScript code based on the input.

In this demo, we’re adding an event handler to a <button> element using the HTML attribute onclick. Whatever we write in this attribute gets interpreted as JavaScript and runs when we click the element.

There are many event handlers we can use. You can find some common ones below for mouse-based events. Try changing onclick in the demo with one of these!

  • onclick
  • onmouseenter
  • onmouseleave
  • onmouseover
  • onmousedown
  • onmouseup
  • onwheel
