Week 1 Splitting things up

The last demo was pretty boring. While it was technically an HTML file, it wasn’t doing anything special. To do anything cool with HTML, we need to start creating HTML elements.

The most basic HTML element is the <div>. Almost every HTML element is indicated by an opening and closing tag, so to create a div we need to start with <div> and end with </div>.

But this is still BORING. To make our code interesting, we’ll need to start using a different programming language in combination with HTML. That language is CSS, and it controls the styling of HTML elements.
