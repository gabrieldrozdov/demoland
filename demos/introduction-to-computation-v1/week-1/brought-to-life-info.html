Week 1 Brought to life

OK, so we used code to make something. Yay!

But our code kind of ended up just like a normal ddocument. What can code do that a normal document can’t do? Here’s one thing: it can move!

In this demo, you’ll find a single div that’s changing styles automatically. This is accomplished by using an @keyframes animation. CSS animations let use queue up a whole bunch of CSS property-value combinations and have them automatically animate between each other.

For this pair programming challenge, your task is to convey a specific genre with your single div and a keyframed animation. Pick from one of the following genres:

  • Comedy
  • Action
  • Horror
  • Romance
  • Mystery
  • Fantasy
  • (something of your own choosing!)

This time, the element’s position property is set to absolute. This let’s us move the element anywhere we want on the page by using the top and left properties (or the bottom and right properties).

CSS Properties

Here are the new CSS properties for animations:

  • animation-name
  • animation-duration
  • animation-iteration-count
  • animation-direction
  • animation-timing-function (find more timing functions here!)
  • animation-fill-mode
  • animation-delay

Here are the CSS properties for absolute positioning:

  • position
  • top
  • left
  • bottom
  • right

Here are the CSS properties for all HTML elements:

  • width
  • height
  • margin
  • padding
  • background-color
  • border
  • border-radius
  • box-shadow
  • transform
