Absolute Positioning CSS Self-portraits

Try creating a self-potrait using HTML elements and CSS properties! Aspire to be Francine but settle wherever you end up.

Things to try out during this demo:

  • Try creating new elements and classes to create your self-portrait.
  • Try using different CSS units and seeing how the elements change size when you change your screen size.
  • See if you can understand what the box-sizing property does.

Things used in this demo:
