Week 2 Variables

Variables can be written using const, let, or var.

You can redefine variables defined used let or var, but const variables can’t be changed, which means they are immutable.

When we define a variable for the first time, we MUST use const, let, or var. If we are redefining a mutable variable (let or var), we just need to say the variable’s name and set it to a new value.
