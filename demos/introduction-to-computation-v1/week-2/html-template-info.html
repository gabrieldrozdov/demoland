Week 2 HTML template

Our demos so far have been a little simplified. Let’s make them a more realistic by using a proper HTML template.

This demo features some new concepts:

  • We have new meta elements that help a web browser render our page.
  • The <head> element contains things like the title of our site (which appears in the browser tab) as well as our CSS, which goes inside the <style> element.
  • The <body> element contains contains our visible content as well as our JavaScript, which goes inside the <script> element.
  • There are three different ways to write comments, which are invisible code descriptions. The notation depends on which language/syntax you’re following (HTML, CSS, and JavaScript).

Most demos will use this template from now on.
