Week 2 Recap

Last week, we played around with HTML and CSS, exploring how designing with code is different from traditional design. We went pretty far, but we’re going to pivot to JavaScript this week. So here’s what you should remember from that class for now:

  • HTML elements have opening and closing tags
  • CSS works by combining properties with values
  • We write CSS in the <style> element
  • We can combine multiple CSS styles in a class, indicated by a period followed by a name and curly brackets
  • We connect CSS classes with HTML elements using the class attribute, which lives inside of an element’s opening tag
