Interacting with type

When do you interact with text on the web? When the text is a link!

Your Challenge

Find a variable font you want to use on Google Fonts. Make sure to filter by variable only!

Then, import your font into this demo. You can do that by pressing “Get font” in Google Fonts, then pressing “Get embed code,” and lastly copying the <link...> embed code.

Once you’ve done this, you’ll be able to use your font in your file! Specify its name in the font-family CSS property, and then use the font-variation-settings property to create an animated link!
