Animating Graphics A story of a square

Not everything on the web is text. How can we bring a basic shape to life?

In this demo, you’ll find a more generic HTML element — a <div> — that we can use to do basically anything. So, let’s try moving it around with some new CSS properties and giving it a personality!

Your challenge

In this demo, we have a 10s animation with keyframes at every 10% interval. How can you give a story to this shape through basic CSS animations?

Try to create a narrative that evokes one of the following genres:

  • Comedy
  • Action
  • Horror
  • Romance
  • Mystery
  • Fantasy
  • (something of your own choosing!)

CSS Properties

Here are the CSS properties for styling shapes:

  • top
  • left
  • width
  • height
  • background-color
  • opacity
  • border
  • border-radius
  • box-shadow
  • filter
  • transform

And here are the new CSS properties for animations:

  • animation-name
  • animation-duration
  • animation-iteration-count
  • animation-direction
  • animation-timing-function (find more timing functions here!)
  • animation-fill-mode
  • animation-delay
