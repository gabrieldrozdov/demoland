Variable Fonts Let’s vary it up!

Websites are really good at animating variable fonts. Once we import a custom font file using the @font-face rule, we can simply set its variable axes using a normal CSS property.

To find a font’s variable axis names and range of values, try plugging it into fontgauntlet.com. You can also play with variable fonts on the Variable Fonts Workshop website!
