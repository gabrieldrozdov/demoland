HTML and CSS

HTML consists of elements, which start with an opening tag (like <p>) and end with a closing tag (like </p>). HTML handles the content of your website.

CSS consists of classes, properties, and values. CSS classes get added inside the opening tag for HTML elements (like <p class="_">). Then, we can style our HTML element by adding CSS properties to that class.
