Keyframed CSS Animating text

As you’ve just experienced, designing a website means setting CSS properties for specific classes and elements. While this can be confusing at first, it has a convenient side effect — to animate CSS, you just tell your code what property values to animate to!

To do this, we create an @keyframes rule that defines an animation. This animation interpolates between multiple CSS properties. We can then assign that animation and customize it using a variety of new CSS properties.

CSS Properties

Here are the new CSS properties for animations:

  • animation-name
  • animation-duration
  • animation-iteration-count
  • animation-direction
  • animation-timing-function (find more timing functions here!)
  • animation-fill-mode
  • animation-delay
