Getting Started Styling text

In this demo, you’ll find a poem typeset in HTML and styled using CSS. Before we get into animating text, let’s try to write some basic code!

Your challenge

Modify the CSS classes to change the styling of the text. How can you use CSS properties to make the text better represent the content of the poem?
