Type as Tool Toggling Variable States

So far, we’ve looked at using CSS to control the look of our variable fonts. We can also use JavaScript to dynamically set CSS values.

In this demo, we have a button that uses the onclick="_" attribute. This lets us run a JavaScript function when user clicks the button. In this case, that function is called “toggleMyVariableClass”.

Our function does two things:

  1. Create a variable using var="_" that corresponds to the HTML element with id="my-element". We can do this using document.querySelector("_").
  2. Add or remove a class from the element we just targeted. This is accomplished via _.classList.toggle("_"). In this case, we’re toggling the class “serif”. That means that if the element doesn’t have the class, the class is added. And, if the element does already have the class, then the class is removed.

If we want to add or remove a class manually, we can also use _.classList.add("_") and _.classList.remove("_").

This is also a good time to introduce two ways of finding and fixing bugs in your code: the alert(_) and console.log(_) functions. alert(_) creates a popup dialog in your web browser, which the user is able to see. console.log(_) outputs an invisible log in your browser’s console, which is visible via the inspect window (Mac: COMMAND+OPTION+I). You can access a light version of the browser console via the button in the bottom-left corner of the Editor panel.
