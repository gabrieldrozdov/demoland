Let’s say we have a few text elements. How can we group them together?

We can use the generic div element to do this!

Divs let us group a whole bunch of elements. In this example, we would call the outer div the parent and the inner paragraphs children.

Parents pass down certain properties to their children. This is called inheritence. For example, the border and padding properties don’t get passed down, but the font-size property does.

Divs don’t need to contain any content. We can also play around with them to use them purely graphically.
