But what about JavaScript? What does it do? Do we need it?

Some coders will tell you that coding in HTML and CSS isn’t real code. And while they’re annoying and rude for saying that, here’s what they’re trying to say:

HTML and CSS languages designed for the web and really only applicable to websites. Meanwhile, JavaScript works like 99% of the programming languages that exist. If you learn JavaScript, you will easily be able to learn how to code in any language.

For example, here’s a website that shows how you can code the same algorithm into hundreds of different languages, including JavaScript. Notice that the site does not offer translations for HTML and CSS, because those languages cannot code algorithms.

So, I can teach you the gist of HTML and CSS in an hour or so, but I can really only teach you JavaScript over the course of a semester (or two). If you want to learn JavaScript, here’s what you should know:

  • How to create and modify variables of different data types
  • How to use arithmetic, comparison, and logical operators
  • How to use conditionals (if/else statements)
  • How to work with randomization
  • How to write and apply functions
  • How to use “for” and “while” loops
  • How to manipulate a website using the Document Object Model (DOM)
  • How to add interactivity using event listeners

Luckily for you, there’s a Cheatsheet page that links to resources for learning about all of these topics!

For now, let’s just have some fun and see what JavaScript can do!
