The previous demo showed that code can get pretty messy pretty fast. Luckily, we can organize CSS separately from HTML so that it’s easier to tell the two apart.

To add CSS to our site, let’s use the style element. This lets us apply CSS to every element of a particular type (and also separate our code onto multiple lines).

But what if we only want to apply CSS to a couple of paragraphs and not every single one? Then, we can use CSS classes!

CSS class names start with a period and can’t have any spaces. If you use punctuation, you should only uses dashes/hyphens.

Notice how we overrided the font-family property with our class. CSS will override other CSS if it’s more applied in a more specific way to an element. So, we applied CSS to ALL paragraph elements, and then applied a CSS class to only a few elements.

So in order of specificity, this paragraph gets its CSS from all paragraphs, and then from its class, and then from its inline CSS.
