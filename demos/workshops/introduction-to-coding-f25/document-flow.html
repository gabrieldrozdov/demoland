One additional way to break elements out of the flow of the page is to use absolute positioning. This completely removes an element from the flow of the page and puts it whereever we want it to go.

Whenever we position elements on the page, we need to think about the CSS box model. This indicates that every HTML element gets rendered as a box with some standard properties, like margin, padding, border, background, etc.
