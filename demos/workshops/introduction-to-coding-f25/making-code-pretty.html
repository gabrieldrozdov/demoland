HTML is not very useful by itself. We need to use CSS to actually start making things look interesting.

We can apply CSS to an HTML element by adding the style attribute inside of the opening tag of the element. This is called inline CSS.

We write CSS by first writing a property name followed by a value. So, the property here is “color” and the value is “red” (a default CSS color).

If we want to apply multiple CSS properties to an element, we have to separate them via semi-colons.

By default, HTML elements are going to keep being added to the bottom of the page and stretch the full width of the page. We can use CSS to change how they’re sized and laid out.

We can push an element to the right side of the page by setting its left margin to “auto”.

Combining left and right margin gives us a centered element.

Just like with HTML elements, how is anyone supposed to know what CSS properties exist?
