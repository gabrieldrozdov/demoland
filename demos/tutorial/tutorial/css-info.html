The Basics What is CSS?

CSS (Cascading Style Sheets) is a programming language that lets you control the appearance (or “style”) of HTML elements. Its code looks a little different than HTML. These differences are called a programming language’s syntax.

CSS consists of properties with values organized into classes. We can then apply those classes to HTML elements.

We write CSS classes inside the <style> element, which we typically place at the top of our document. If our CSS is not inside of the <style> element, then the browser will interpret it as HTML and the website won’t render properly.

Another way to write CSS is by writing inline CSS, which lives directly inside of HTML tags via the style="_" attribute.

HTML handles content and CSS handles design. To make a site fully interactive, we’ll need to use JavaScript. Open the next demo to learn about JavaScript!
