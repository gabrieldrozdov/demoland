What is CSS?

CSS (Cascading Style Sheets) is a programming language that lets you control the appearance (or style) of HTML elements. Its code looks a little different than HTML. These differences are called a programming language’s syntax.

Coding in CSS

We write CSS classes inside the <style> element, which we typically place at the top of our document. If our CSS is not inside of the <style> element, then the browser will interpret it as HTML and the website won’t render properly.

Properties, values, and classes

CSS consists of properties with values. A property controls some aspect of what an element looks like, such as its font or color. We write the property name, followed by a colon :. Then, we assign a value to that property, followed by a semi-colon ;.

We organize these property-value pairs inside of classes. To create a class, we write a period . and then a name, which cannot include any spaces.

We then apply these classes to HTML elements by adding a class attribute inside of the element’s opening tag. This follows the format class=" ", where the class name goes inside of the quotes.

We can also apply CSS to a specific HTML element using inline CSS. This makes use of the style=" " attribute, which allows us to write CSS properties and styles directly within the HTML element.
