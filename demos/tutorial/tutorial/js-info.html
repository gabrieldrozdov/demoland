What is JavaScript?

While HTML and CSS let us organize and style content, JavaScript (or JS) gives our site deeper interactivity and functionality. This functionality has a wide range of possibilities, from creating basic text pop-ups (like the one featured in this demo) to much more complicated interactions (like this entire site’s interface and code editor).

JavaScript is very different from HTML/CSS and works more like a traditional programming language. That means that what you code in JavaScript is also possible in virtually any other programming language, while the same isn’t true for HTML and CSS.

A warning: be sure not to confuse JavaScript with Java, which is an entirely unrelated programming language! Java was very popular, so the creator of JavaScript named it after Java to try to capitalize on that popularity.

Coding in JS

We write JavaScript inside of the <script> element, which we typically place at the bottom of our document.

While websites will often run fine even if there are a couple of typos in our HTML and CSS, JavaScript is much more finicky and can easily break a site if it contains typos.

Functions

JavaScript can get pretty complicated, but generally speaking it involves creating functions that do stuff.

Once we create a function, we need to run it! In this demo, we’re using the onclick=" " HTML attribute, which lets us run a JavaScript function when the element is clicked on. (This is called an inline event handler.) Try clicking on the button in the Preview panel and see what happens!
