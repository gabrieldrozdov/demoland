The Basics What are JavaScript Libraries?

JavaScript can get pretty complicated. Sometimes, implementing a feature using JavaScript can require tens of thousands of lines of code. Luckily, thereâ€™s a long history of developers working for the greater good of the community and releasing complicated coding projects for free.

JavaScript Libraries are a way for developers to release their code for others to use. In many cases, JavaScript libraries give us new functions to simplify tasks that would otherwise be extremely difficult or tedious.

We can include a JavaScript library in our site by giving the <script> element the src="_" attribute and specifying a URL that links to the library we want to use. We typically put this element at the top of our document. We can either download the library and include it in the files for our website, or embed it via a Content Delivery Network (CDN) link.
