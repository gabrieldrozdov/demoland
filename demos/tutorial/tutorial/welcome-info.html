Welcome to DEMOLAND

DEMOLAND is a collection of coding demos that are editable right inside your web browser. Learn how to code without the need to download any additional software!

Interface

Info panel

This is the first panel, which you’re currently reading. This panel provides information about your current demo.

Editor panel

This is the second panel, which is where you can view and edit code. The bottom of the Editor panel lets you customize the Editor’s interface, as well as control how often your code gets previewed.

Preview panel

This is the third panel, which displays a live preview of your code. The bottom of the Preview panel displays the current dimensions of the Preview panel.

Customizing the interface

Hide/show panels using the three buttons to the center-left. Resize panels by dragging between them.

Navigation

Site organization

DEMOLAND is organized into books containing chapters of demos. Right now, you are in the “Tutorial” book’s “How to use DEMOLAND” chapter.

Navigating between demos

To view all the demos in the current chapter, click on the menu bar at the top of the Info panel. You can also quickly navigate to the next or previous demo using the arrow buttons at the top of the Info panel.

Opening a different chapter

On the top of this page, you’ll find your current book’s title and its chapters. Click on the book title to see an overview of its chapters, and click on a chapter name to jump to that chapter.

Opening a different book

Return to DEMOLAND’s homepage to open a different book. You can do this by clicking the buttons in the top-left and bottom-left corners.

Controlling the current demo

Reset the current demo, download its code, or open the demo in a new tab using the buttons at the bottom-right.

Getting lost?

You can reopen this tutorial at any time by clicking the ? button at the bottom-left.

Writing Code

Write your code directly in the Editor panel. All demos are presented as single HTML files.

HTML

By default, all code is interpreted as HTML.

CSS

Add CSS using the <style> element.

JavaScript

Add JavaScript using the <script> element.

Open the next demo in this chapter to learn more about how to write HTML, CSS, and JavaScript!
