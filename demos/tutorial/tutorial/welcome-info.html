Welcome to DEMOLAND!

DEMOLAND is a like a bookstore, except instead of reading books, you’re reading code! This site features demos that’ll teach you how to code websites using HTML, CSS, and JavaScript.

Site navigation

DEMOLAND is organized into books with multiple chapters. Each chapter contains one or more demos.

Basic Info

View your current book, chapter, and demo in the top-left corner of the screen.

Switching Demos

Use the yellow buttons in the top-right corner to change the current demo or chapter.

Current Demo Controls

Use the red buttons in the bottom-right corner to reset the current demo, download its code, or open it in a new tab.

Lost?

Use the purple buttons in the bottom-left corner to go to the homepage, re-open this tutorial, or open up a blank demo. You’ll also find a link to Cheatsheet, which contains a (mostly) complete list of HTML, CSS, and JavaScript concepts!

Viewing and editing demos

The interface features three main panels.

Press the buttons in the bottom-center of the screen to hide/show these panels.

Resize the panels by dragging the vertical bars between them.

Info Panel

This is the panel you’re reading right now. The Info panel provides information about the current demo.

Editor Panel

This is the panel in the middle. The Editor panel lets you view and edit the code for the current demo.

Preview Panel

This is the rightmost panel. The Preview panel shows you a preview of your code as it would appear on a live website.

Writing Code

All demos on DEMOLAND use the web’s three main programming languages: HTML, CSS, and JavaScript. By default, the editor interprets all code as HTML, but we can add CSS and JavaScript inside of an HTML file.

HTML (Hypertext Markup Language) is used to organize content, like text or images.

CSS (Cascading Style Sheets) is used to style that content.

JavaScript (or JS) is used to add deeper functionality to websites.

The next few demos in this chapter showcase how HTML, CSS, and JavaScript use different syntax. “Syntax” is the set of rules you need to follow when coding in a specific programming language. Open the next demo in this chapter to learn more about HTML!

Shortcuts

Here are a few helpful shortcuts for when you’re editing code:

  • Command+] and Command+[, or Control+] and Control+[ to indent/outdent the selected lines of code.
  • Command+/ or Control+/ to comment in/out the selected lines of code.
  • Control+Space to bring up an auto-complete dialog.
  • Hold Command or Control and click in several locations to type in multiples spots at once.
