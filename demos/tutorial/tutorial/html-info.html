What is HTML?

HTML (Hypertext Markup Language) is the main programming language of the web. We use HTML to organize a website’s content, like its text, images, videos, etc.

Coding in HTML

The code in the Editor panel to the right is all HTML. Web browsers interpret HTML code and turns it into real websites. For example, HTML tags are not visible on a live website, which you can see in the Preview panel.

We write code using plain text. That means that our code does not feature any styling information you’d normalling get in a document editor, like font, alignment, or color. Instead, the code is automatically assigned colors based to indicate if it is written correctly or contains typos.

Elements

When writing HTML, we create elements, which are marked by opening and closing tags. For example, a paragraph opens with <p> tag and closes with </p> tag.

Check out Cheatsheet for a (mostly) full list of HTML elements!
